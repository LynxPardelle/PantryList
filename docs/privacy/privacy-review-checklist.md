# Privacy Review Checklist

Use this checklist when a change touches AI/OCR, receipt/photo capture, household sharing, temporary links, sessions/devices, waste tracking, payments, billing, subscriptions, sponsored offers, or partner/retailer handoff.

## Required Checks

- Data touched: list the household, identity, session/device, waste, receipt, photo, payment, or sharing data involved.
- User visibility: state what the user sees before data is created, shared, uploaded, or deleted.
- Consent: state whether the action is opt-in, automatic, or required for the feature.
- Retention: state how long data is kept and how users can delete or export it.
- Access control: state who can read, mutate, revoke, or delete the data.
- External processors: name every external provider that receives data.
- Abuse case: state the main misuse path and mitigation.
- Rollback: state how the feature can be disabled or reverted.
